Hallo Sammlergemeinde! :-)

Ich möchte euch gerne einmal die Varianten der Pappagalos aus Mexiko vorstellen.
Diese Varianten werden den meisten Sammlern unbekannt sein, da sie soweit ich weiss nirgends genau beschrieben werden.
Deshalb habe ich mir die Mühe gemacht und versucht möglichst gute Fotos hinzukriegen, die die Unterschiede verdeutlichen sollen.
Für mich ist es einer der interessantesten Sätze meiner recht umfangreichen Auslandssammlung.
Ich hoffe euch gefällt der Bericht.Falls jemand Verbesserungsvorschläge oder Korrekturen hat, so kann er sie mir gerne Mitteilen.

Auf der Bildern ist die Reihenfolge immer:
"Los Pappagalos" (Italien) links ; "Piratoquet" (Frankreich) mitte ; "Loropiratas" (Mexiko) rechts

Generell ist zu sagen, dass die italienischen und französischen Figuren eine "© Ferrero" Kennung haben 
und die Mexikanischen Figuren haben "© MPG" Kennungen (TT301, TT302, TT303, TT306, TT307, TT308, TT309).
Desweiteren sind die Brauntöne der einzelnen Länder unterschiedlich, was man gut auf Bild 7 erkennen kann.
Die braune Farbe findet sich an: 
beiden Holzbeinen,Bodenbrettern der Möwe, Kufen der Babyschaukel, Korken der Flasche!
Die genaueren Unterschiede werde ich nun auflisten:

Bild 1 + 2 (JackO'Tarl) : 
- Italien: Schnabel und Füsse gelb; Flasche mit Aufschrift (leicht Marmoriert) ; Hut blau ; Jacke lila
- Frankreich: Schnabel und Füsse gelb ; Flasche mit Aufschrift (leicht Marmoriert) ; Hut blau ; Jacke rot
- Mexiko: Schnabel und Füsse orange ; Flasche ohne Aufschrift (einfarbig ) ; Hut dunkelblau ; Jacke dunkelrot

Bild 3 -8 (RobySpugna) :
- Italien: Schnabel und Füsse gelb ;Möwe orange (Grundmaterial matt) ; Boden 3tlg. dünn ; Kopftuch lila
- Frankreich: Schnabel und Füsse gelb ; Möwe orange (Grundmaterial matt) ; Boden 3tlg. dünn ; Kopftuch rot
-Mexiko: Schnabel und Füsse orange ; Möwe dunkelorange (Grundmaterial glänzend) ; Boden 2tlg. dick ; Kopftuch dunkelrot

 Bild 9 +10 (Capitan Jellino):
- Italien: Schnabel gelb ; Halstuch blau ; Hut blau 
- Frankreich: Schnabel gelb ; Halstuch rot ; Hutblau 
- Mexiko: Schnabel orange ; Halstuch dunkelrot ; Hut dunkelblau 

Bild 11 + 12 (Bebe' Piratina) : 
-Italien: Schnabel und Füsse gelb ; Knopf blau ; Häubchen rosa ; Schaukel Grundmaterial dunkelbraun ; Fahne Grundmaterial hellbraun
-Frankreich: Schnabel und Füsse gelb ; Knopf rot ; Häubchen rosa ; Schaukel Grundmaterial dunkelbraun ; Fahne Grundmaterial hellbraun
- Mexiko: Schnabel orange ; Knopf dunkelrot ;Häubchen pink ; Schaukel Grundmaterial hellbraun ; Fahne Grundmaterial dunkelbraun 

Bild 13 - 16 (Gaia Lavandaia) :
- Italien: Schnabel und Füsse gelb ; Haare Kupferfarben ; Hut lila ; Eimer mit Henkeln, Schaum nur vorne (dick)
- Frankreich: Schnabel und Füsse gelb ; Haare Kupferfarben ; Hut rot ; Eimer mit Henkeln, Schaum nur vorne (dick)
- Mexiko: Schnabel und Füsse orange ; Haare dunkelbraun-metallic ; Hut dunkelrot ; Eimer ohne Henkel, Schaum vorne und hinten (dünn)

Bild 17 + 18 (José Lustraben) :
- Italien: Schnabel und Füsse gelb ; Kopftuch lila ; Flasche marmoriert
- Frankreich: Schnabel und Füsse gelb ; Kopftuch rot ; Flasche marmoriert
- Mexiko: Schnabel und Füsse orange ; Kopftuch dunkelrot ; Flasche einfarbig

Bild 19 + 20 (Nuto Chiedeaiuto) :
- Italien: Schnabel und Füsse gelb ; Kopftuch lila ; Haifisch ohne Eckzähne ; Wasser blau
- Frankreich: Schnabel und Füsse gelb ; Kopftuch rot ; Haifisch ohne Eckzähne ; Wasser blau
- Mexiko: Schnabel und Füsse orange ; Kopftuch dunkelrot ; Haifisch mit Eckzähnen ; Wasser durchsichtig
